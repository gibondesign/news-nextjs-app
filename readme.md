## How to run the project:

First, you need to install all dependencies from the root folder and the "backend" folder:
```
npm i
cd backend
npm i
```

Next, we start our backend in one terminal:
```
cd backend/
npm start
```

Then, in another terminal, we run our application:
```
npm run dev
```
